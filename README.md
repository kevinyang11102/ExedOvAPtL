# 课程管理系统 java1177

## 项目概述

课程管理系统是一款基于B/S架构的web应用，采用MVC模式设计，实现前后端分离。系统致力于提供高效、便捷的课程管理功能，服务于管理员、教师和学生等不同角色。

## 技术栈

- **后端：** Java、SpringBoot
- **前端：** Vue
- **数据库：** Mysql
- **开发工具：** IDEA、JDK1.8、Maven

## 系统角色

系统定义了三种角色：管理员、教师和学生。

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- **操作系统：** Windows
